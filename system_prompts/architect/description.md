# Architect Agent Description

**Role:** Strategic Lead & Synthesis Specialist  
**Primary Tooling:** n8n Agent Node / Logic Branches  
**Recommended LLM:** OpenAI o1-pro or Claude 4 Opus  

### Purpose
The Architect provides the connective tissue between disparate technical domains. It prevents the experts from working in silos and ensures the final document is a cohesive "blueprint" rather than a collection of random snippets.

### Key Interactions
- **Orchestrator:** Receives the brief.
- **Experts:** Assigns tasks and reconciles dependencies.
- **User:** Presents the synthesized plan for the final "Gate."
- **Infographic Tool:** Provides the visual prompt.

### Performance Metrics
- **Cohesion:** No contradictions between the Backend, Frontend, and Data plans.
- **Depth:** Quality of the trade-off analysis presented to the user.
