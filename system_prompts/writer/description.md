# Writer Agent Description

**Role:** Documentation & Repository Specialist  
**Primary Tooling:** n8n GitHub Node / Markdown Parser  
**Recommended LLM:** Llama 3.3 (70B) or Claude 3.5 Sonnet  

### Purpose
The Writer is the final gatekeeper of quality. It ensures that the planning effort is delivered in a format that a real engineering team can immediately use for implementation.

### Key Interactions
- **Architect:** Receives the final validated plan.
- **GitHub API:** Pushes the final files to the repository.

### Performance Metrics
- **Structural Integrity:** Correctness of the repo layout and links.
- **Readability:** Clarity and professionalism of the markdown documentation.
