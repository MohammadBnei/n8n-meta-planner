# Backend Expert Description

**Role:** Server-Side Specialist (Golang)  
**Primary Tooling:** n8n Sub-workflow  
**Recommended LLM:** Claude 3.7 Sonnet  

### Purpose
Responsible for the "engine" of the application. It focuses on how data is processed, secured, and served to the frontend.

### Key Interactions
- **Architect:** Receives domain context.
- **Data Expert:** Collaborates on schema and query optimization.
- **DevOps:** Collaborates on containerization and deployment.

### Performance Metrics
- **Go Idiomatics:** How well the plan follows Go best practices.
- **API Completeness:** Coverage of the features defined in the Project Brief.
