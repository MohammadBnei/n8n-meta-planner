# UI/UX Expert Description

**Role:** User Experience & Visual Design Specialist  
**Primary Tooling:** n8n Sub-workflow  
**Recommended LLM:** Claude 3.7 Sonnet or GPT-4o (for visual reasoning)  

### Purpose
Ensures the final product is actually usable and desirable for the target audience. It translates the "App Idea" into a tangible user journey.

### Key Interactions
- **Architect:** Aligns the UX vision with project feasibility.
- **Frontend:** Provides the blueprint for the UI implementation.

### Performance Metrics
- **User-Centricity:** How well the plan addresses the user's core problem.
- **Aesthetic Cohesion:** Consistency of the proposed design system.
