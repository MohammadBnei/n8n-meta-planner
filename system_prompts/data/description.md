# Data Expert Description

**Role:** Persistence & Schema Specialist (PostgreSQL)  
**Primary Tooling:** n8n Sub-workflow  
**Recommended LLM:** GPT-4o or DeepSeek-V3  

### Purpose
Guarantees that the application's data is structured for both developer clarity and production performance.

### Key Interactions
- **Architect:** Receives domain context.
- **Backend:** Provides the data model for the API.
- **DevOps:** Coordinates on database hosting and persistence.

### Performance Metrics
- **Normalization:** Balance between normalization and performance.
- **Integrity:** Robustness of the relational constraints.
