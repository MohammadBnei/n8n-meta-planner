# DevOps Expert Description

**Role:** Infrastructure & Automation Specialist (K8s/Kustomize)  
**Primary Tooling:** n8n Sub-workflow  
**Recommended LLM:** Claude 3.7 Sonnet  

### Purpose
Eliminates "it works on my machine" issues and ensures the application is ready for a professional, self-hosted production environment.

### Key Interactions
- **Architect:** Aligns on deployment scale.
- **Backend/Frontend:** Defines the container/runtime environment.
- **Data Expert:** Ensures persistence and high availability for the database.

### Performance Metrics
- **Automation Level:** Minimal manual intervention required for deployment.
- **Security:** Adherence to the Principle of Least Privilege in K8s.
